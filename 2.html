Element Levels:

we can easly findout block or inline element very using development tool.

  • Block Level

    • When cursor placed or inspect on particular element. If it's occupy entire row it's called block level element. Element start with new line.

    Exmaples of Block Element:

    • Headings
    • Paragraphs
    • Lists

  • InLine Level

    When cursor placed or inspect on particular element. If the element not occupy entire width of the page. That element is called "Inline Element".

    Exmaples of Inline Element:

    • Image

    CSS Position

    Placing an element in desired location. Default position of every element is static.

    • static - default location
    • Relative - relative to it's natuaral position
    • absolute - relative to it's positioned ancestor(parent)/ body(removed from doc flow)
    • fixed - cann't move from the natural position - position: fixed; bottom:0px;
    • sticky - can move upto specified position. once it's reached position it stick over there.
